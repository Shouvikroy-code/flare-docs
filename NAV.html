Help Option in Navigation Menu

With this enhancement, we have added the Help option in the navigation menu of the HealthRules MRF Suite. This new feature aims to enhance user experience by providing easy access to the latest MRF documentation.

Feature Details:

  • Hyperlink to Documentation: The Help option includes a hyperlink to the latest MRF documentation, which can be accessed at HealthEdge Helpcenter.

  • Standard Help Icon: The Help option is represented by the standard Help icon, accompanied by a callout that reads "Help".

  • New Tab Opening: Clicking the Help icon will open the documentation page in a new tab on the browser.

[MRF-231, MRF-218EPIC].
